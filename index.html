关于这个项目

这是一个机器学习入门展示项目。模型使用 librosa 从音频中提取 MFCC、色度、频谱质心、频谱带宽等特征， 再用 scikit-learn 的随机森林（Random Forest）进行分类训练。 前端通过 Flask 后端调用模型，实时返回每个乐器类别的预测概率。

项目适用于学习 ML 完整流程（数据准备 → 特征工程 → 模型训练 → Web 部署）， 而非追求工业级识别精度。
